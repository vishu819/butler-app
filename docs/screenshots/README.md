# Screenshots

Drop PNG screenshots here with these exact filenames so the main README renders them:

| File | Screen | How to capture |
|------|--------|----------------|
| `home.png` | Home — charcoal level card, lime "Today's session" card, goals | Log in → Home tab |
| `session.png` | A session question (MCQ + follow-up + deeper MCQs) | Home → tap "Today's session" |
| `progress.png` | Progress — level, streak/rating, dark skill-map bars | Progress tab |
| `practice.png` | Practice — speed-MCQ category picker | Practice tab |
| `others.png` | Others menu (AI News, Papers, Articles, Library, Account) | Others tab |
| `papers.png` | Must-read papers with summarize-on-open | Others → Must-read Papers |
| `login.png` | Login / sign-up screen | Log out |

Tip: use a mobile viewport (~375×812) in your browser's device toolbar — Butler is mobile-first. Keep images ~600px wide so they render cleanly on GitHub.
